# SecureStream - Proyecto Final POO
**Autor:** Jair Naranjo  
**Fecha:** Marzo 2026  

## Objetivo
Plataforma de streaming con validación de seguridad RBAC y servicios API JSON integrados con MySQL.

## Funcionalidades
- **8 Servicios Web:** Gestión de catálogo, usuarios, auditoría y login.
- **Persistencia:** Conexión real a base de datos MySQL.
- **Frontend:** Dashboard dinámico diseñado con Bulma CSS.

## Instrucciones
1. Ejecutar Laragon/MySQL.
2. Ejecutar `go run main.go`.
3. Usuario Admin: `admin` | Clave: `1234`
